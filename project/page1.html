🧬 Criar Ativo BBP

Defina a identidade raiz do ativo. O DNA_IDENTITY será gerado e registrado no Bitcoin.

📌 DNA_IDENTITY (Identidade do Ativo)

Segundo o padrão BBP-0001: BBP:id:<domain>:<namespace>:<identifier>

Organização ou coleção responsável pelo ativo
Nome único dentro do namespace
🔍 Seu DNA_IDENTITY será: BBP:id:art:exemplo:exemplo
📄 Template: NFT
Você será redirecionado para o template específico após criar o DNA_IDENTITY.
