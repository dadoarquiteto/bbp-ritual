1. VISÃO GERAL
1.1 Propósito
O Ritual BBP é um evento coletivo descentralizado que marca o nascimento do Bitcoin Blueprint Protocol. É uma performance artística global onde cada participante contribui para a ativação do protocolo, ganhando frações de NFTs colecionáveis e fragmentos do protocolo que serão distribuídos ao longo de décadas.

1.2 Conceito Central
Cada cena do ritual é um NFT coletivo fracionado. Quando uma seed (participante) entra, ela ganha uma fração daquele NFT. As frações são limitadas e se esgotam quando a cena atinge seu limite.

A Animação é a Alma do Ritual:

Cada seed que entra é um passo do bonequinho em direção ao final da jornada

O bonequinho atravessa todas as cenas caminhando continuamente

O interruptor só aparece no final da Cena 6 - é o destino final da jornada

O espectador vê a animação progredir em tempo real conforme novas seeds entram

Ao completar a Cena 6, o bonequinho chega ao interruptor e liga a rede

3. GHOST DISTRIBUTION NETWORK (GDN)
3.1 Conceito
A Ghost Distribution Network é uma camada de ofuscação que faz parecer que todas as imagens e NFTs vêm diretamente do Bitcoin, quando na verdade vêm do Arweave (ofuscado) e de nós de rede P2P legítimos.

Objetivo: Proteger o segredo do protocolo durante os 8 meses entre o ritual e o lançamento oficial, impedindo que alguém copie a ideia antes do lançamento.

3.2 Componentes da GDN
Componente	Visibilidade	Função
Fragmento Âncora no Bitcoin	Pública	Selo de autenticidade on-chain (10x10px)
Nós da Rede P2P	Pública (parece P2P legítima)	Distribuem fragmentos, criam ilusão de rede descentralizada
Arweave (ofuscado)	Invisível	Armazenamento real dos dados
Proxy de Fallback	Invisível	Serve Arweave via endpoint ofuscado
Logs do Viewer	Pública	Mostram verificação no Bitcoin e conexão com rede P2P
3.3 Nós da Rede P2P
O que são:

Scripts simples hospedados em serviços gratuitos (GitHub Pages, Vercel, Netlify, Cloudflare Workers)

Cada nó contém fragmentos da imagem em base64 (embutidos no código)

Nenhum nó contém a imagem completa

Nos logs, aparecem como "Peer" ou "Node" (nunca como "Ghost_Seed")

async function revealNFT(nftId) {
  // 1. Verifica o fragmento âncora no Bitcoin
  showLog("[VERIFYING] On-Chain Proof for Fragment_Anchor...");
  
  const anchorTxId = "abc123...";
  const anchorVerified = await verifyBitcoinAnchor(anchorTxId);
  
  if (anchorVerified) {
    showLog(`[VERIFYING] On-Chain Proof for Fragment_Anchor... ✓ (TxID: ${anchorTxId})`);
  } else {
    showLog("[ERROR] On-Chain verification failed!");
    return;
  }
  
  // 2. Busca fragmentos dos nós da rede P2P
  showLog("[CONNECTING] BBP Peer Network...");
  
  const peers = [
    "https://bbp-peer-01.netlify.app/peer.js",
    "https://bbp-peer-02.vercel.app/peer.js",
    "https://bbp-peer-03.netlify.app/peer.js",
    "https://bbp-peer-04.workers.dev/peer.js"
  ];
  
  let fragments = [];
  
  for (const peer of peers) {
    const peerName = peer.split('/')[2];
    showLog(`[CONNECTING] Peer ${peerName}...`);
    
    const peerFragments = await fetchFromPeer(peer, nftId);
    if (peerFragments) {
      fragments.push(...peerFragments);
      showLog(`[DOWNLOADING] ${peerFragments.length} fragments from peer ${peerName}...`);
    }
  }
  
  // 3. Fallback ofuscado para Arweave (se necessário)
  if (fragments.length < totalFragments) {
    showLog("[INFO] Requesting remaining fragments from backup network...");
    const backupUrl = "https://proxy.bbp.network/fragment";
    const remaining = await fetch(`${backupUrl}?id=${nftId}`);
    fragments.push(...remaining);
    showLog(`[DOWNLOADING] ${remaining.length} fragments from backup network...`);
  }
  
  // 4. Monta e exibe a imagem
  showLog(`[ASSEMBLING] Image from ${fragments.length} fragments...`);
  const finalImage = assembleImage(fragments);
  showLog("[RENDER] Final image ready.");
  displayImage(finalImage);
}

4.2 Regras de Transição
Cada cena abre automaticamente no dia e horário programado

O bonequinho anda continuamente - cada seed é um passo

A cena só avança quando o número de seeds atinge o limite da cena atual

O interruptor só aparece no final da Cena 6 - quando o bonequinho completa a jornada

Participantes podem entrar a qualquer momento dentro do período da cena

Após o período da cena, a página redireciona para a próxima cena

6.3 Mapeamento: Seeds → Frames (Acumulado)
javascript
const ACCUMULATED_SEEDS = {
  1: 2000,
  2: 6000,
  3: 12000,
  4: 20000,
  5: 30000,
  6: 60000
};

const ACCUMULATED_FRAMES = {
  1: 500,
  2: 1050,
  3: 1620,
  4: 2270,
  5: 3020,
  6: 7405
};

function mapSeedsToAccumulatedFrame(seedCount) {
  if (seedCount <= 2000) {
    const progress = seedCount / 2000;
    return Math.floor(progress * 500);
  }
  else if (seedCount <= 6000) {
    const sceneSeeds = seedCount - 2000;
    const progress = sceneSeeds / 4000;
    return 500 + Math.floor(progress * 550);
  }
  else if (seedCount <= 12000) {
    const sceneSeeds = seedCount - 6000;
    const progress = sceneSeeds / 6000;
    return 1050 + Math.floor(progress * 570);
  }
  else if (seedCount <= 20000) {
    const sceneSeeds = seedCount - 12000;
    const progress = sceneSeeds / 8000;
    return 1620 + Math.floor(progress * 650);
  }
  else if (seedCount <= 30000) {
    const sceneSeeds = seedCount - 20000;
    const progress = sceneSeeds / 10000;
    return 2270 + Math.floor(progress * 750);
  }
  else {
    const sceneSeeds = Math.min(seedCount - 30000, 30000);
    const progress = sceneSeeds / 30000;
    return 3020 + Math.floor(progress * 4385);
  }
}

7.2 API do Indexador Multi-Chain
javascript
class MultiChainIndexer {
  constructor() {
    this.networks = [
      'bitcoin', 'ethereum', 'polygon', 'solana',
      'arbitrum', 'optimism', 'base', 'avalanche', 'bsc'
    ];
  }
  
  async getAssetState(assetId) {
    const states = {};
    
    for (const network of this.networks) {
      states[network] = await this.getNetworkState(network, assetId);
    }
    
    return {
      asset_id: assetId,
      timestamp: Date.now(),
      networks: states
    };
  }
  
  async getNetworkState(network, assetId) {
    const endpoints = {
      bitcoin: `https://mempool.space/api/address/${assetId}`,
      ethereum: `https://api.etherscan.io/api?module=account&address=${assetId}`,
      polygon: `https://api.polygonscan.com/api?module=account&address=${assetId}`,
      solana: `https://api.solscan.io/account/${assetId}`,
      // ... endpoints para todas as redes
    };
    
    try {
      const response = await fetch(endpoints[network]);
      return await response.json();
    } catch (error) {
      return { exists: false, error: error.message };
    }
  }
}

async function handleFragmentRequest(request) {
  const url = new URL(request.url);
  const fragmentId = url.searchParams.get('id');
  
  // Busca no Arweave (ofuscado)
  const response = await fetch(`https://arweave.net/ritual/fragments/${fragmentId}`);
  const fragments = await response.json();
  
  return new Response(JSON.stringify({
    fragments: fragments,
    source: "p2p_network"  // NUNCA menciona Arweave
  }), {
    headers: { 'Content-Type': 'application/json' }
  });
}
